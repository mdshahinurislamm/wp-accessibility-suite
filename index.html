Standards

Built for real compliance

WP Accessibility Suite helps your site meet international WCAG 2.1 standards — from baseline Level A all the way to full AAA conformance with PRO.

A
Level A — Baseline
The minimum standard. Covers skip links, image alt text, form labels, keyboard access, and basic ARIA landmarks. Included in the free version.
AA
Level AA — Recommended
The legal standard in most countries. Adds contrast ratios, focus visibility, consistent navigation, and more. Fully supported in both free and PRO versions.
AAA
Level AAA — Gold Standard
The highest level. Includes enhanced contrast, sign language, no timing, and location breadcrumbs. PRO's audit engine helps you identify and fix AAA gaps.

FAQ

Common questions

Does it work with my theme? ↓
Yes. The plugin injects its toolbar and styles independently of your theme via wp_footer. It works with any theme including Divi, Elementor, Avada, GeneratePress, Astra, and block themes.
Will it slow down my site? ↓
No. The CSS and JS are small (under 30 KB combined) and load in the footer. There are no external API calls on page load. The AI alt text only runs in the admin.
Is it GDPR compliant? ↓
Completely. All visitor preferences are stored in their own browser's localStorage. Nothing is sent to our servers. No cookies. No tracking without the optional analytics feature explicitly enabled.
Can I install PRO without the free version? ↓
Yes — PRO is a standalone plugin. Install it instead of the free version, not alongside it. It includes all free features and more.
What's the AI alt text API key for? ↓
You provide your own OpenAI API key. The PRO plugin sends images to OpenAI's Vision API to generate descriptive alt text. Typical cost is less than $0.01 per image.
Do I get updates after purchase? ↓
Yes. Your license includes all future updates for the license term. Agency licenses include lifetime updates. Single-site licenses receive updates for one year, with renewal at a discounted rate.
